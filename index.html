TypeScript Haqida

TypeScript - bu JavaScript-ning kengaytirilgan versiyasi bo'lib, tiplarga asoslangan qo'shimcha xususiyatlarni taqdim etadi. U katta loyihalarda kodni samarali boshqarish va xatolarni oldindan aniqlash imkonini beradi. Statik tiplarni qo'llab-quvvatlaydigan, ommabop va kuchli tip tizimiga ega bo'lgan dasturlash tili. TypeScript, JavaScript'ning barcha imkoniyatlarini o'z ichiga oladi va unga qo'shimcha ravishda ba'zi yangi xususiyatlar ham beradi. TypeScript, JavaScript dasturlarini ishlab chiqishda aniqlikni oshiradi va kengaytirilgan tahlilni amalga oshirishga imkon beradi.

Asosiy xususiyatlar:
  • Statik tip tizimi: TypeScript kodni yozish jarayonida o'zgaruvchilar va funksiyalar uchun tiplarni belgilash imkonini beradi. Bu kompilyatsiya vaqtida xatoliklarni aniqlashga yordam beradi va kodni tushunishni osonlashtiradi.
  • Transpilyatsiya: TypeScript kodini JavaScript kodiga transpilyatsiya qilish orqali ishlatiladi. TypeScript fayllari .ts kengaytmasiga ega bo'lib, ularni JavaScript kodiga .js kengaytmasiga aylantirish kerak. Bu jarayonni TypeScript transpiler (tsc) amalga oshiradi.
  • Yaxshi xatoliklarni aniqlash: Statik tip tizimi yordamida xatoliklar kompilyatsiya vaqtida aniqlanadi, bu esa kodni ishlab chiqish jarayonini osonlashtiradi va tezlashtiradi.
  • Obyektga yo'naltirilgan dasturlash (OOP): TypeScript Java, C#, va boshqa OOP tillariga o'xshash sinflar (classes), meros olish (inheritance), polimorfizm (polymorphism), interfeyslar (interfaces) kabi imkoniyatlarni taqdim etadi.
  • Type Inference (tipni aniqlash): TypeScript ba'zi o'zgaruvchilarni yoki ifodalarni tiplashni avtomatik ravishda amalga oshirishi mumkin. Misol uchun, agar siz sonni o'zgaruvchi sifatida ishlatsangiz, TypeScript ularning tipini avtomatik tarzda aniqlaydi.

  • Any Type

    Any tipi TypeScript-da istalgan turdagi qiymatni saqlash uchun ishlatiladi. U JavaScript-ning dinamik tipiga o'xshaydi. 

    let anyValue: any = 10;
anyValue = "Hello";
anyValue = true;
anyValue = [1, 2, 3];

  • Union Type

    Union tipi bir nechta turlardan birini qabul qilishi mumkin bo'lgan o'zgaruvchilarni aniqlash imkonini beradi. 

    let id: number | string;
id = 101;
id = "TS101";

  • Tuple Type

    Tuple tipi massivning har bir elementi uchun aniq turni belgilash imkonini beradi. 

    let person: [string, number, boolean];
person = ["John", 30, true];

  • Enum Type

    Enum tipi o'zaro bog'liq konstantalar to'plamini yaratish uchun ishlatiladi. 

    enum Color {
  Red,
  Green,
  Blue
}
let c: Color = Color.Green;

  • Void Type

    Void tipi odatda hech qanday qiymat qaytarmaydigan funksiyalar uchun ishlatiladi. 

    function logMessage(message: string): void {
  console.log(message);
}

  • Function Type

    Function tipi funksiyaning parametrlari va qaytarish qiymatining turlarini aniqlash uchun ishlatiladi. 

    let add: (x: number, y: number) => number;
add = (a: number, b: number) => a + b;

  • Interface

    Interface (interfeys) â€” bu obyektnig tuzilmasini belgilash uchun ishlatiladi. 

    interface Person {
    name: string;
    age: number;
}
let person: Person = { name: 'Alice', age: 30 };

  • Classes

    TypeScriptda sinflar (classes) yordamida ob'ektga yo'naltirilgan dasturlashni amalga oshirish mumkin. 

    class Person {
    name: string;
    constructor(name: string) {
        this.name = name;
    }
    greet() {
        console.log('Hello, ' + this.name);
    }
}
let person = new Person('Alice');
person.greet();  // Output: Hello, Alice
